PROYECTOS

Aquí encontrarás mis proyectos universitarios y estudios de caso correspondientes a cada uno

CATSINO

Catsino es un proyecto universitario que simula un casino temático de gatos. El objetivo del proyecto fue desarrollar una plataforma interactiva donde los usuarios pudieran participar en diversos juegos de casino: Blackjack, Bingo, Mayor o menor y Senku.

Se implementaron interfaces frontend personalizadas, utilizando tecnologías web para ofrecer una experiencia visual atractiva y dinámica. Además, se utilizo Java y Spring Framework en el backend, para manejar la lógica de los juegos y la interacción con el servidor.

VER MÁS

MYSTERYQUIZ

Mystery Quiz es un innovador proyecto universitario que combina la diversión del juego de trivia con un enfoque educativo. Los jugadores se enfrentan a una serie de preguntas intrigantes que desafían su conocimiento en diversas categorías. El objetivo es responder correctamente para avanzar y subir de nivel, lo que hace que cada sesión de juego sea única y emocionante.

Este proyecto fue desarrollado utilizando PHP para la lógica del servidor y Mustache para el renderizado de las vistas, lo que permite separar la lógica de presentación y facilitar la creación de interfaces limpias y dinámicas.

VER MÁS

BOOTIFY

Botify es un proyecto universitario desarrollado en .NET que consiste en un bot que recomienda musica en base al estado de animo del usuario.

El proyecto se centra en la creación de asistentes virtuales personalizados. Se priorizó una arquitectura robusta que combina un backend escalable con interfaces intuitivas.
