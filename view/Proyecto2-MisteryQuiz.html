Sobre el Proyecto: MisteryQuiz

MisteryQuizMisteryQuiz es una plataforma de preguntas y respuestas interactivas diseñada para ofrecer entretenimiento educativo, fomentar la competitividad y gestionar contenido de manera eficiente. Este proyecto combina una experiencia de usuario intuitiva con roles bien definidos para mantener la calidad del contenido y la administración de la plataforma.

Flujo del usuario

  • Registro e Inicio de Sesión : Los usuarios deben registrarse para acceder al juego. El registro incluye la creación de un perfil personalizado que les permitirá rastrear su progreso y visualizar estadísticas
  • Perfil del Jugador: Cada usuario cuenta con un perfil donde se reflejan sus puntos acumulados y datos personales.

Dinámica del juego

  • El núcleo del juego son cuestionarios interactivos compuestos por preguntas con cuatro opciones de respuesta.
  • Cada respuesta correcta otorga puntos que se suman al perfil del jugador.
  • Las preguntas están categorizadas, lo que permite ajustar la dificultad y el tipo de contenido ofrecido.
  • El sistema ofrece retroalimentación inmediata sobre las respuestas, potenciando el aprendizaje continuo.

Gestión de Roles

1.Administrador

Objetivo:Supervisa el contenido, controla la actividad en la plataforma y estadisticas

2. Editor

Objetivo:Encargado de crear, revisar y actualizar las preguntas del juego.
