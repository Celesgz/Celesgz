Sobre el Proyecto: Botify

Botift. Este proyecto consiste en el desarrollo de un chatbot musical diseñado para ofrecer una experiencia personalizada a los usuarios. El chatbot utiliza la API de Spotify para proporcionar recomendaciones de canciones, álbumes y artistas basadas en el estado de ánimo y las preferencias musicales del usuario. También permite la creación y gestión de listas de reproducción, así como el acceso a información detallada sobre artistas y álbumes.


El proyecto se basa en diversas tecnologías, incluyendo la API de Spotify para acceder a un amplio catálogo musical, el Microsoft Bot Framework para la creación del chatbot, y ASP.NET Core como backend. Se emplea también JSON Web Tokens (JWT) para la autenticación de usuarios y JavaScript junto con AJAX para una interfaz interactiva. Para el diseño responsivo, se utiliza Bootstrap.

Beneficios esperados

El chatbot mejorará la experiencia del usuario al interactuar con la música, ofreciendo una herramienta valiosa para descubrir nuevos artistas y mantener un vínculo más cercano con su música favorita. Al integrar tecnologías modernas y una interfaz intuitiva, el chatbot buscará atraer y retener a los usuarios, convirtiendo la música en una experiencia social y personalizada.
